À propos de MdpGen

MdpGen est un gestionnaire de mots de passe pas comme les autres. Contrairement aux gestionnaires traditionnels, MdpGen ne stocke aucun de vos mots de passe, ce qui signifie qu'il n'y a aucun risque de fuite de données.

MdpGen utilise une approche unique pour générer vos mots de passe : il combine le nom du site et un code secret unique (que vous utilisez pour tous vos sites) pour produire un mot de passe sécurisé.

Le mot de passe généré sera toujours le même pour un site donné, tant que vous utilisez le même code secret. Ainsi, vous n'avez besoin de mémoriser qu'un seul code secret pour accéder à tous vos sites.

L'algorithme utilisé par MdpGen est basé sur le hachage cryptographique SHA-3, conçu par la NSA. Cet algorithme est spécifiquement conçu pour résister à toute tentative d'inversion du calcul, ce qui garantit la sécurité de vos mots de passe.

En utilisant MdpGen, vous pouvez être sûr que vos mots de passe sont non seulement sécurisés, mais aussi totalement privés, car ils ne sont jamais stockés nulle part.
