À propos de MdpGen

Do you use the same password for all your online accounts? Don’t want to give your passwords to password managers without knowing what they do with them? Looking for a cross-platform tool you can use anywhere, without any installation or registration?

Welcome to the world of MdpGen, a password manager whose sole purpose is to guarantee your digital security. Imagine a world where your passwords are both unbreakable and tamper-proof, while remaining as easy to manage as counting to three.

Unlike traditional password managers that store your sensitive information on centralized servers—easy targets for hackers—MdpGen takes a radically different approach. With MdpGen, there is nothing to store, nothing to steal. You’ve never been safer.

How does it work? It’s simple: MdpGen uses a generation algorithm based on only two elements—the name of the website or service you have an account on, and a unique secret code known only to you. These two elements are then combined to create a secure password that you can confidently use on the specified site. The generated password will therefore always be the same for a given site, as long as you use the same secret code. Flawless security, without compromise.

At the heart of MdpGen lies the cryptographic hash algorithm SHA-3, designed by security experts at the NSA. SHA-3 is the ultimate defense against malicious attacks. Your passwords are protected by one of the most advanced technologies in the world. MdpGen runs SHA-3 sixty thousand times in a row, ensuring that a brute-force attack would take at least 170 billion years on a standard computer.

With MdpGen, you can finally say goodbye to fears about data leaks and intrusions. Your security is in your hands—literally. Take back control of your passwords with MdpGen.

How do you start using MdpGen? Choose a secure secret code that you can easily remember. Then begin replacing all your passwords with those generated by MdpGen. To obtain them, enter the name of the site you want to log into, followed by your secret code. Once all your passwords have been changed, every time you want to log into a website or application, you just repeat the same process. Type the site name and your secret code, and MdpGen will always return the correct password.

Keep in mind that MdpGen is a password generator—in other words, it stores absolutely nothing. That’s why if you make a mistake in your secret code or the site name, MdpGen won’t warn you and will still return a corresponding password. For the site name, it’s therefore recommended to use only lowercase letters and separate words with hyphens, so you never make a mistake when entering it again later.

Finally, although MdpGen is an online tool, you can absolutely use it offline. Either your browser allows you to save pages for offline use, or you can download the MdpGen HTML page and reopen it locally. Indeed, MdpGen fits into a single HTML file. To download the HTML page, right-click on the page and select “Save As.”
